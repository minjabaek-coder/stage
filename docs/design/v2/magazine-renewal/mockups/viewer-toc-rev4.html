변경 요약

  • 목차 = 하단 가로 필름스트립 (데스크톱·모바일 공통) — 기존: 데스크톱 우측 세로 패널 / 모바일 하단 시트(세로, 화면 가림) → 하단 가로 밴드. 오버레이 아님 — 열리면 페이지 영역이 줄어 페이지 전체가 항상 보임, 닫으면 복귀. 썸네일 + 쪽번호만(제목 생략), 현재 페이지 골드, 좌→우 스크롤·현재 항목 자동 센터.
  • 모바일 확대 버튼 — 제거 (핀치 줌 제스처 유지)
  • 모바일 넘김 효과 — 항상 즉시 전환 → 넘김 버튼 제거
  • 모바일 진입 — 자동 전체화면 시도(사용자 제스처 제약 시 폴백)
  • 모바일 하단 컨트롤 — 목차 · 다크 · 전체 (3개)
  • 모바일 하단 진행률 = 드래그 스크러버 — 바를 드래그/탭하면 해당 페이지로 이동(시크, 손잡이 표시). 상단 진행률(골드 언더라인)은 표시 전용.
